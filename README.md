# Pokemon Custom Battle Simulator

ブラウザで遊べるシンプルなポケモン風ターン制バトルシミュレーターです。プレイヤーと相手のチームをそれぞれ 3 体選び、タイプ相性・交代・状態異常を使いながら対戦できます。

## 実装済み機能

- ターン制バトル
- 技ダメージ計算
- タイプ相性反映
- プレイヤー / 相手チームのポケモン選択 UI
- バトルログ表示
- 状態異常（やけど、どく、もうどく など）
- 交代、まもる、ステルスロック、どくびし
- ランダム選出と簡易 AI

## セットアップ

```bash
npm install
```

## 起動手順

### 開発用サーバーを起動

```bash
npm start
```

起動後、ブラウザで以下を開いてください。

```text
http://127.0.0.1:4173
```

## ビルド

静的配布用ファイルを `dist/` に出力します。

```bash
npm run build
```

## テスト

```bash
npm test -- --runInBand
```

## 遊び方

1. 自分と相手のチームをそれぞれ 3 体選びます。
2. 「バトル かいし!」で対戦を開始します。
3. 「たたかう」で技を使い、「ポケモン」で交代します。
4. 各技ボタンにはタイプと相性ヒントが表示されます。
5. ログ欄でターンごとの行動結果を確認できます。
