# オリジナルポケモンデザイン案

## 1. 基本情報
- **名前**: ルミヴェール
- **分類**: きょくこうポケモン
- **タイプ**: こおり / フェアリー
- **高さ**: 1.4m
- **重さ**: 32.0kg
- **特性**:
  - ゆきがくれ
  - ひかりのベール（隠れ特性）

## 2. デザインコンセプト
ルミヴェールは、オーロラと薄い氷のヴェールをモチーフにしたポケモンです。
体のシルエットはシンプルな四足獣ベースですが、背中から尾にかけて半透明の氷膜が重なり、
見る角度で色が変化する“偏光”の印象を強く持たせます。

### ビジュアルの要点
1. **主配色**: 白銀 + ラベンダー + ミントブルー
2. **アクセント**: 目元と尾先に薄い金色
3. **質感**: 体毛はマット、氷膜はグロス
4. **形状言語**: 攻撃的な尖りを減らし、流線とレイヤーで神秘性を表現

## 3. 生態・世界観
- 極夜の地域に生息し、月明かりを受けると背中の氷膜が発光する。
- 吹雪の夜には仲間同士で円陣を組み、光のカーテンを作って外敵を惑わせる。
- 人前に姿を現すことは稀だが、迷子を安全な場所まで導いたという伝承がある。

## 4. バトル上の個性（イメージ）
- **役割**: サポート寄りの特殊アタッカー
- **得意戦術**: 天候（ゆき）と壁技を活用した盤面コントロール
- **専用技案**: `オーロラステップ`
  - タイプ: フェアリー（特殊）
  - 威力: 75 / 命中: 100
  - 追加効果: 30%で相手のすばやさを1段階下げる

## 5. デザイン制作ガイド（実装向け）
- **シルエット優先**: モノクロでも「背中のヴェール」で識別できること。
- **アニメーション**: 待機時に氷膜がゆっくり波打ち、攻撃時に偏光が強まる。
- **UIアイコン**: 顔アップより、尾から背中のヴェールを含む斜め構図が映える。

## 6. バリエーション案
- **色違い**: ラベンダーを深い群青へ、金色アクセントを銀色へ変更。
- **リージョンフォーム案**: みず / フェアリー化し、氷膜が液晶のような水膜表現になる。

---
この案は「神秘性」「視認性」「ゲーム内での役割」の3点を同時に成立させることを狙っています。
